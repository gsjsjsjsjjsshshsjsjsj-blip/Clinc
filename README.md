# نظام المواعيد الطبية - شفاء

نظام شامل لإدارة المواعيد الطبية مبني بتقنيات الويب الحديثة مع واجهة مستخدم متجاوبة وواجهة خلفية آمنة.

## 🌟 المميزات الرئيسية

### للمرضى
- **البحث عن الأطباء**: البحث المتقدم حسب التخصص والموقع والسعر
- **حجز المواعيد**: حجز مواعيد بسهولة مع اختيار الوقت المناسب
- **إدارة المواعيد**: عرض وإدارة جميع المواعيد (القادمة، المكتملة، الملغية)
- **الإشعارات الذكية**: إشعارات فورية لتأكيد الحجز والتذكيرات
- **الملف الشخصي**: إدارة البيانات الشخصية وتغيير كلمة المرور

### للأطباء
- **إدارة المواعيد**: عرض وإدارة مواعيد المرضى
- **تحديث الحالة**: تغيير حالة المواعيد (مؤكد، مكتمل، ملغي)
- **الملف المهني**: إدارة المعلومات المهنية والتخصصات

### للمديرين
- **إدارة النظام**: إدارة المستخدمين والأطباء والتخصصات
- **التقارير**: إحصائيات شاملة عن النظام

## 🛠 التقنيات المستخدمة

### الواجهة الأمامية (Frontend)
- **HTML5**: هيكل الصفحات
- **CSS3**: التصميم والتنسيق
- **Bootstrap 5.3.3**: إطار العمل للتصميم المتجاوب
- **JavaScript (ES6+)**: التفاعل والوظائف الديناميكية
- **Bootstrap Icons**: الأيقونات

### الواجهة الخلفية (Backend)
- **PHP 8+**: لغة البرمجة الخلفية
- **PDO**: للتعامل مع قاعدة البيانات
- **Prepared Statements**: للحماية من هجمات SQL Injection
- **RESTful API**: واجهات برمجية منظمة

### قاعدة البيانات
- **MySQL 8+**: قاعدة البيانات الرئيسية
- **UTF-8**: دعم كامل للغة العربية
- **Foreign Keys**: العلاقات بين الجداول
- **Indexes**: تحسين الأداء

## 🚀 التثبيت والإعداد

### المتطلبات
- PHP 8.0 أو أحدث
- MySQL 8.0 أو أحدث
- خادم ويب (Apache/Nginx)
- Composer (اختياري)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/your-username/medical-appointment-system.git
cd medical-appointment-system
```

2. **إعداد قاعدة البيانات**
```sql
-- إنشاء قاعدة البيانات
CREATE DATABASE medical_appointments CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- استيراد هيكل قاعدة البيانات
mysql -u root -p medical_appointments < database/schema.sql
```

3. **تكوين قاعدة البيانات**
```php
// تعديل ملف config/database.php
private $host = 'localhost';
private $db_name = 'medical_appointments';
private $username = 'your_username';
private $password = 'your_password';
```

4. **إعداد الخادم**
```apache
# Apache .htaccess
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^api/(.*)$ api/$1 [L]
```

5. **الوصول للنظام**
```
http://localhost/medical-appointment-system/
```
